# Elijah Farrell's Portfolio Website

This repository contains the source code for my personal portfolio website, showcasing my projects, experience, skills, and education.

## Features

- Responsive design for optimal viewing on any device.
- Dark mode and customizable theme colors.
- Interactive typewriter effect.
- Real-time comment section powered by Firebase.

## Technologies Used

- HTML, CSS, JavaScript
- Firebase Firestore
- Font Awesome Icons

## View Live

The site is available live via GitHub Pages: [Visit Portfolio](https://elijah-farrell.github.io/portfolio/)
